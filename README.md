# 🇮🇳 Tribute Page – Major Mohit Sharma

This is my first FreeCodeCamp **Responsive Web Design Project** – a tribute page dedicated to **Major Mohit Sharma**,
Ashoka Chakra awardee and Indian Army Para Special Forces officer.  

---

## 🪖 About Major Mohit Sharma
- Born: **13 January 1978**, Rohtak, Haryana  
- Officer in **1 PARA (Special Forces)**  
- Conducted daring undercover operations in Kashmir (2004)  
- Martyred on **21 March 2009** in Kupwara while fighting terrorists  
- Awarded **Ashoka Chakra (Posthumous)**, India’s highest peacetime gallantry award  

---

## 💻 Project Features
- Clean, responsive design  
- Timeline of Major Mohit Sharma’s life  
- Tribute quote & external Wikipedia link  
- Built with **HTML5 + CSS3**  
- Deployed on **Netlify** for free hosting
